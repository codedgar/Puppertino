Getting started

Puppertino is a straightforward framework to use as you would expect. You can use Puppertino in two ways:

Download the full CSS.

Downloading the full CSS is recommended if you are going to use several components of Puppertino or you are planning on using them. However, if you are going to use if some of them we recommend you to go to the next way of using Puppertino.

Download Puppertino

Using CSS based on components.

You can also browse the Puppertino Repo to find whatever components you want to use.

Browse repo

What about JS?
